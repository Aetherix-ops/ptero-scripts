## About

ptero-scripts is a growing collection of battle-tested scripts designed to make managing Pterodactyl Panel easier. Scripts are written in multiple languages тАФ each chosen for what it does best.

> This repository is separate from [ptero-eggs](https://github.com/Aetherix-ops/ptero-eggs).
> - **ptero-eggs** тАФ JSON egg configurations imported into Pterodactyl Panel
> - **ptero-scripts** тАФ Scripts run on the host server by the admin

## Installation

Clone this repository:

    git clone https://github.com/Aetherix-ops/ptero-scripts.git
    cd ptero-scripts

Make bash scripts executable:

    chmod +x */*.sh

Each script has its own config file. Edit before running:

    nano <script-folder>/config.sh     # bash scripts
    nano <script-folder>/config.py     # python scripts
    nano <script-folder>/config.js     # node.js scripts
    nano <script-folder>/config.env    # go scripts
